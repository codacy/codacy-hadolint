## Pin versions in apt get install.

### Problematic code:

```dockerfile
FROM busybox
RUN apt-get install python
```

### Correct code:

```dockerfile
FROM busybox
RUN apt-get install python=2.7
```
### Rationale:

https://docs.docker.com/develop/develop-images/dockerfile_best-practices/#apt-get
