## Delete the apt-get lists after installing something.

### Problematic code:

```dockerfile
RUN apt-get update && apt-get install --no-install-recommends -y python
```

### Correct code:

```dockerfile
RUN apt-get update && apt-get install --no-install-recommends -y python \
 && apt-get clean \
 && rm -rf /var/lib/apt/lists/*
```

### Rationale:

https://docs.docker.com/develop/develop-images/dockerfile_best-practices/#apt-get

> In addition, when you clean up the apt cache by removing /var/lib/apt/lists it reduces the image size, since the apt cache isnâ€™t stored in a layer. Since the RUN statement starts with apt-get update, the package cache is always refreshed prior to apt-get install.

### Notes:

Clean up must be performed in the same RUN step, otherwise it will affect image size.
